# Architecture Decision Records

This directory records version-level architecture decisions for Possum.

Every completed Possum version should have an ADR that captures:

- the version identifier and date
- status
- context
- accepted decisions
- consequences and tradeoffs
- verification evidence
- key commits
- follow-up decisions deferred to later versions

Use monotonically increasing ADR numbers:

```text
0001-possum-v1-local-customer-simulator.md
0002-possum-v2-<short-topic>.md
```
